# TxtCraft Public Patents 📜⚓

Welcome to the **TxtCraft Public Patents** repository. This is the official public registry where users of the [TxtCraft](https://txtcraft.com) platform can openly publish their intellectual property, ideas, and patent drafts.

## 🔗 Blockchain Proof of Existence

Every document published in this repository is designed to be paired with an **OpenTimestamps (.ots)** proof. 

TxtCraft utilizes the Bitcoin blockchain to cryptographically anchor documents. This process creates an immutable, timestamped mathematical proof that a specific document existed at a specific point in time, without revealing the document's content on the blockchain itself.

## 🕵️‍♂️ How to Verify a Patent

If you have downloaded a Markdown patent file (`.md`) and its corresponding proof file (`.ots`) from a TxtCraft user, you can easily verify its authenticity and anchoring date:

1. **Using TxtCraft:** Go to the Verification Portal on the TxtCraft platform, upload both the `.md` and `.ots` files, and our system will validate the cryptographic hash against the Bitcoin blockchain.
2. **Using OpenTimestamps (Independent Verification):** You can independently verify the proof by visiting [OpenTimestamps.org](https://opentimestamps.org/) and dragging both files into their verification tool.
