# txtcraft Publications 📜⚓

The official public registry of [txtCraft](https://txtcraft.com/) 

## 🔗 Blockchain proof of existence

Each document is paired with an OpenTimestamps (`.ots`) proof. 

txtcraft anchors a cryptographic hash to the Bitcoin blockchain — an immutable, timestamped proof that a document existed at a given time, without revealing its content.

## 🕵️ Verify a document

With a document (`.md`) and its proof (`.ots`):

1. **On txtcraft** — upload both files to the Verification Portal.
2. **Independently** — drop both files into [OpenTimestamps.org](https://opentimestamps.org/).
